Robotvoet (SE&A)

  • Tech Stack: Raspberry Pi, Stepper Motor, HTML, CSS, Javascript, API
  • Github URL: Project Link | Publication
Waar gaat het project over?

Dit project is een onderdeel van het vak "system enginering and analysis". Het doel van dit project is om kinderen van middelbare leeftijd te laten zien dat wiskunde leuk kan zijn. Dit moesten we doen door gebruik te maken van voetbal en wiskunde. Daarom hebben we een spel gemaakt waarbij de kinderen een balletje moeten schieten op een doel. Dit wordt gedaan door middel van een spel waarbij de kinderen een aantal vragen moeten beantwoorden. De vragen zijn gebaseerd op de wiskunde die ze op school leren. We hebben er ook verschillende niveau's in gemaakt zodat het voor iedereen leuk is om te spelen.

Wat heb ik gedaan?

Ik heb dit project gemaakt met 2 andere studenten. Ik heb me vooral bezig gehouden met het maken van de constructie en op het einde de styling van de webpagina. We hebben dit project gemaakt met behulp van een raspberry pi die verbonden is met een steppermotor. aan de steppermotor zit een constructie vast die het balletje kan schieten. Daarnaast hebben we onder de voet een bewegende plaat waarmee je de richting van het schieten kan bepalen.

Conclusie

Dit project was een heel leuk project om te maken. Omdat we echt een moesten kijken wat onze opdrachtgevers wilden hebben. Op de laatste dag hebben we dan nog heel veel extra dingen toegevoegd die de opdrachtgevers ons gegeven hadden tijdens de feedback. Ik wil daarom ook de opdrachtgevers bedanken voor de feedback die ze ons gegeven hebben. Maar zeker ook mijn groepsgenoten Wim en Samir voor het harde werk dat ze geleverd hebben.
